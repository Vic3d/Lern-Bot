# 🚀 Lernbot — Vercel Deploy (wie Josh Dashboard)

**Genau wie das Josh Dashboard — Live URL in 2 Min!**

---

## **Schritt 1: Vercel Account**

1. Geh zu https://vercel.com
2. Sign up mit GitHub (oder Email)
3. Autorisiere Vercel für dein GitHub

---

## **Schritt 2: Deploy (Copy-Paste)**

**Terminal / PowerShell öffnen:**

```bash
# 1. Zum Repo gehen
cd /pfad/zum/lernbot

# 2. Login bei Vercel
vercel login

# 3. Deploy!
vercel deploy --prod
```

Dann fragt Vercel:
```
? Set up and deploy "/pfad/zum/lernbot"? [Y/n] y
? Which scope should contain your project? [dein-github]
? Project name? lernbot
? Automatically update setting to match detected framework Next.js? [Y/n] y
? Create outgoing sourceMap files? [y/N] n
```

---

## **Schritt 3: URL bekommen**

Nach ~30 Sekunden kriegst du:

```
✅ Production: https://lernbot.vercel.app
```

**Das ist deine Live URL!**

---

## **Alternative: Via GitHub (noch einfacher)**

1. Push zu GitHub (schon gemacht ✅)
2. Geh zu https://vercel.com/new
3. Import GitHub Repo: `Vic3d/Lern-Bot`
4. Click "Deploy"
5. Done! — URL kommt automatisch
